### C2. Chart motion (guarded, tasteful)

Add a draw‑in animation to the polygon and a fade‑in to markers, respecting reduced motion.

```tsx
// in Radar.tsx
const pathRef = useRef<SVGPathElement>(null);
useEffect(() => {
  const prefersReduced = window.matchMedia('(prefers-reduced-motion: reduce)').matches;
  if (prefersReduced || !pathRef.current) return;
  const len = pathRef.current.getTotalLength();
  pathRef.current.style.strokeDasharray = `${len}`;
  pathRef.current.style.strokeDashoffset = `${len}`;
  requestAnimationFrame(() => {
    pathRef.current!.style.transition = 'stroke-dashoffset 900ms cubic-bezier(.2,.8,.2,1)';
    pathRef.current!.style.strokeDashoffset = '0';
  });
}, []);
```

*Rationale:* Subtle, purpose‑built motion signals innovation without gimmicks; must respect `prefers-reduced-motion`. ([brand.mit.edu][1])

## D) Micro‑interactions & states

### D1. Count‑up stats in the hero

Animate “500+ teams” / “78% improved ROI” when entering viewport.

```tsx
function useCountUp(target: number, dur = 1200) {
  const [n, setN] = useState(0);
  useEffect(() => {
    let raf: number, start = performance.now();
    const tick = (t: number) => {
      const k = Math.min(1, (t - start)/dur);
      setN(Math.round(target * (1 - Math.pow(1 - k, 3)))); // ease-out
      if (k < 1) raf = requestAnimationFrame(tick);
    };
    raf = requestAnimationFrame(tick);
    return () => cancelAnimationFrame(raf);
  }, [target, dur]);
  return n;
}
```

Wire this up only after an IntersectionObserver marks the stat as visible; **skip** if reduced‑motion.
